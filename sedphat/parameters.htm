parameters:

 

there are the global parameters, which are dependent on the analysis model chosen

local parameters are dependent on the experiment type, and are specific for each experiment.  For example, each experiment with sedimentation data can have specific information if the meniscus or bottom will be treated as floating (local) parameter, or if it's fixed.  Similarly, noise parameters can be specific, as well as the extinction coefficients.

Global parameters are, for example, binding constants, s-values, molar masses.  An additional parameter that occurs with each analysis and is not specified in the global parameters box is the partial-specific volume of the sample.  This can also be a floating parameter (thus permitting Edelstein-Schachman-type analysis with density variation techniques).
