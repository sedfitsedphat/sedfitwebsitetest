Exit

Data | Exit

This closes the program. Results of the data analysis should be stored manually, using either the save fit command, the copy commands, the notebook, or other methods. However, some information on the analysis remains stored on the hard disk, in a ASCII text file named ‘~tmppars.*’ located in the data directory. This file includes some of the fitting parameters and meniscus and bottom location. Also, this file contains information if the program crashed or was properly closed, giving opportunity to restore the analysis after loading of files.  However, this restoration functions properly only in some of the models.
